## 科目B試験（午後試験）出題分野

科目B試験では以下の分野から出題され、情報セキュリティ（必須）+ 選択4問 = 計5問を解答する。

| 分野 | 系統 | 必須/選択 |
|------|------|-----------|
| 情報セキュリティ | テクノロジ | 必須 |
| 経営戦略 | ストラテジ | 選択 |
| プログラミング | テクノロジ | 選択 |
| システムアーキテクチャ | テクノロジ | 選択 |
| ネットワーク | テクノロジ | 選択 |
| データベース | テクノロジ | 選択 |
| 組込みシステム開発 | テクノロジ | 選択 |
| 情報システム開発 | テクノロジ | 選択 |
| プロジェクトマネジメント | マネジメント | 選択 |
| サービスマネジメント | マネジメント | 選択 |
| システム監査 | マネジメント | 選択 |

## 試験概要

| 項目 | 科目A（午前） | 科目B（午後） |
|------|---------------|---------------|
| 出題形式 | 多肢択式（四択） | 記述式 |
| 出題数 | 80問 | 11問 |
| 解答数 | 80問 | 5問 |
| 試験時間 | 150分 | 150分 |
| 合格基準 | 60点以上（100点満点） | 60点以上（100点満点） |

### 科目A 分野別出題数の目安
- テクノロジ系: 約50問
- マネジメント系: 約10問
- ストラテジ系: 約20問

> **備考**: 2026年度（令和8年度）よりCBT方式に移行予定。
> 名称が「午前試験→科目A試験」「午後試験→科目B試験」に変更。
> 出題範囲・出題形式・試験時間に変更なし。
